## About this project

This is a system design project that uses:
- Storybook
- Changesets
- Turborepo
- Tsup
- Typescript
- React
- Vite
- Stitches
- Radix UI


See more at https://joatancavalcante.github.io/storybook-system-design/?path=/docs/home--docs.
